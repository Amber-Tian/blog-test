# 1.登入github 新建一个repo

第一步：创建仓库名称；  
第二步：描述仓库（可以省略）；  
第三步：点击绿色按钮创建仓库。

# 2.进入setting页面设置

第一步： 点击进入创建好的仓库；  
第二步： 点击最右边的setting。

# 3.setting页面找到github pages

第一步：选择master分支；  
第二步：点击save保存；  
第三步：记下链接地址（xxx.github.io/yyy/）。

# 4.将要预览的html路径拼接在链接后面

xxx.github.io/yyy/zzz.html  
自己的用户名,github.io/仓库名/文件路径
